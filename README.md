# 機械学習のサンプル

自己学習用ソースの保管庫です。

## Description

各フレームワークで提供されているチュートリアルやサンプルデータを使った機械学習のサンプルソースを保管。
画像分類、重回帰予測、文章分類など、自分で試したサンプルを掲載。

## Features

気が向いたら勉強する感じで。。。

## Requirement

- Google Colaboratory
- Watson Studio
- Zinrai Deep Learning

## Usage

1. Jupyter Notebookで開いて、
2. Cellをポチポチ押していく。。。

## Anything Else
